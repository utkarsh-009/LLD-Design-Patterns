# About Game and Rules

* Its is a simple two-player paper-and-pencil game.

* Each player selects a piece before the game i.e. 'X' or 'O'.

* Player takes turns placing it on the 3x3 grid.

* A player wins by placing three of its pieces in a row, column, or diagonal.

* The game ends when a player wins or when all grid cells are filled, resulting in a draw.
